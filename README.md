# School Management Swing App

## Description
A Java Swing desktop application for managing school records with user and admin authentication, database integration, and CRUD operations.

## Features
- User & Admin login system
- Add records
- Update records
- Delete records
- Database integration
- GUI using Java Swing

## Technologies Used
- Java
- Swing
- JDBC
- NetBeans
